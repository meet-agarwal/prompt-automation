 Complete PRD: Modular eBay/Etsy HTML Description Template Generator

 Global Constraints

 No JavaScript: All behaviors must be CSS-only.
 CSS Location: All CSS must reside within a single `<style>` tag in the document `<head>`; no inline styles or external stylesheets.
 Responsive Breakpoints: Mobile threshold at 768px.
 Platform Compliance: Only tags and attributes allowed by eBay/Etsy description editors.

 Randomization Directive

For each section, the AI must randomly select exactly one layout variant based on the weights below:

| Section | Layout Variants                                   | Weights                               |
| ------- | ------------------------------------------------- | ------------------------------------- |
| Header  | 4 options                                         | Equal (25% each)                      |
| Banner  | 1 slider layout                                   | 100%                                  |
| Product | 6 options                                         | Equal (16.7% each)                    |
| Tabs    | 1 tabs (desktop) + 1 accordion (mobile)           | 100%                                  |
| Gallery | Static 3×3, Static 5×2, Swipe, Auto‑Slide, Hybrid | Equal     							  |
| Footer  | 3 footer layouts                                  | Equal (33.3% each)                    |
| Theme   | 6 visual themes                                   | Equal (16.7% each)                    |

> Note: Once a theme is chosen, its color palette, fonts, and button styles must apply consistently across all sections.

---

 Document Structure

1. HTML Skeleton with `<head>` containing `<style>…</style>` and `<body>` sections 1–6.
2. Section Containers: Each section wrapped in `<section id="section-{name}">` with two subsections:

    Layout Container: the randomized HTML structure.
    Content Placeholders: a `<div class="content-data">` listing all URLs, texts, and links.

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Product Description Template</title>
  <style>
    / All CSS for layout, themes, responsiveness goes here /
  </style>
</head>
<body>
  <!-- Section 1: Header -->
  <!-- Section 2: Banner -->
  <!-- Section 3: Product -->
  <!-- Section 4: Tabs -->
  <!-- Section 5: Gallery -->
  <!-- Section 6: Footer -->
</body>
</html>
```
